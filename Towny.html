Towny 插件指令

指令 描述
/town显示城镇信息。
/town new [name]创建一个新城镇。
/town delete [name]删除一个城镇（管理员指令）。
/town add [player]邀请玩家加入城镇。
/town kick [player]将玩家移出城镇。
/town set mayor [player]设置新的城镇市长。
/town set name [new name]更改城镇名称。
/town claim为城镇占领当前地块。
/town unclaim取消当前地块的占领。
/town spawn传送到城镇的出生点。
/town deposit [amount]向城镇银行存款。
/town withdraw [amount]从城镇银行取款。
/town set board [message]设置城镇公告板信息。
/town set homeblock设置城镇的中心地块。
/town rank add [player] [rank]为玩家添加城镇头衔。
/town rank remove [player] [rank]移除玩家的城镇头衔。
/nation显示国家信息。
/nation new [name]创建一个新国家。
/nation delete [name]删除一个国家（管理员指令）。
/nation add [town]邀请城镇加入国家。
/nation kick [town]将城镇移出国家。
/nation set king [player]设置新的国家国王。
/nation set name [new name]更改国家名称。
/nation set capital [town]设置国家首都。
/nation ally add [nation]添加盟国。
/nation ally remove [nation]移除盟国。
/nation enemy add [nation]添加敌对国家。
/nation enemy remove [nation]移除敌对国家。
/nation deposit [amount]向国家银行存款。
/nation withdraw [amount]从国家银行取款。
/nation rank add [player] [rank]为玩家添加国家头衔。
/nation rank remove [player] [rank]移除玩家的国家头衔。
/nation spawn传送到国家的出生点。
/resident显示个人信息。
/resident set perm设置个人默认权限。
/resident friend add [player]添加好友。
/resident friend remove [player]移除好友。
/resident friend list列出所有好友。
/plot claim占领当前地块。
/plot unclaim放弃当前地块。
/plot forsale [price]将地块挂牌出售。
/plot notforsale取消地块的出售状态。
/plot set shop将地块设置为商店。
/plot set embassy将地块设置为大使馆。
/plot set arena将地块设置为竞技场。
/plot set farm将地块设置为农场。
/plot set jail将地块设置为监狱。
/plot toggle pvp切换地块的 PvP 状态。
/plot toggle fire切换地块的火焰蔓延状态。
/plot toggle explosion切换地块的爆炸状态。
/plot toggle mob切换地块的怪物生成状态。
/townyadmin访问 Towny 管理员指令。
/townyadmin set mayor [town] [player]设置城镇的市长。
/townyadmin delete town [name]删除一个城镇。
/townyadmin delete nation [name]删除一个国家。
/townyadmin toggle war切换战争模式。
/townyadmin reload重新加载 Towny 插件。
/townyadmin backup创建 Towny 数据备份。
/townyadmin reset重置 Towny 数据（谨慎使用）。
/townyadmin givebonus [town] [amount]为城镇提供额外地块奖励。
/townyadmin tpplot [world] [x] [z]传送到指定地块。
/townyadmin resident [player]管理玩家的 Towny 数据。
/townyadmin town [town]管理城镇的 Towny 数据。
/townyadmin nation [nation]管理国家的 Towny 数据。
